Jsx Fragments ?

  • JSX fragments, represented by <> ... </>, are a way to group multiple JSX elements without introducing an additional parent element.
  • Fragments are useful when you need to return adjacent JSX elements without wrapping them in a div or other container. 

    •             
                
                function MyComponent() { 
                    return (
                      <>
                        <p>Element 1</p>
                        <p>Element 2</p>
                      </>
                    );
                    }
